API backend for Cine Digest

Movies

  • api/v1/searchm/{movie-title}
    search movies by title
  • api/v1/searchm/{movie-title}/{primary-release-year}
    search movies by title and release year
  • api/v1/getm/{movie-id}
    get movie information
  • api/v1/getmr/{movie-id}
    get recommendations based on movie id

TV Shows

  • api/v1/searchs/{show-title}
    search TV shows by title
  • api/v1/gets/{show-id}
    get TV show information
  • api/v1/gets/{show-id}/{season-number}
    get TV show season information
  • api/v1/getsr/{tv-show-id}
    get recommendations based on tv show id
